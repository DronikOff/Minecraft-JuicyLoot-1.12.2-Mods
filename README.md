Гайд по установке:
1. Установить версию майнкрафта forge-optifine 1.12.2
2. Скачать файлы.
3. Удалить все моды из папки mods и config по пути C:\Users\Имя вашего ПК!\AppData\Roaming\\.minecraft
4. Перекинуть установленные файлы,папки: (config, mods, options) по пути C:\Users\Имя вашего ПК!\AppData\Roaming\\.minecraft с заменой!!!
5. Для установки новых модов чтобы не скачивать все моды занаво скачиваем папку mods update и перекидываем папку mods по пути C:\Users\Имя вашего ПК!\AppData\Roaming\\.minecraft если нужно с заменой!!!
