Гайд по установе:
1. Скачать файлы
2. Удалить все моды из папки mods по пути C:\Users\Имя вашего ПК!\AppData\Roaming\.minecraft
3. Перекинуть установленные файлы,папки: (config, mods, options) по пути C:\Users\Имя вашего ПК!\AppData\Roaming\.minecraft с заменой!!!
4. Для установки новых модов чтобы не скачивать все моды занаво скачиваем папку mods update и перекидываем папку mods по пути C:\Users\Имя вашего ПК!\AppData\Roaming\.minecraft если нужно с заменой!!!
